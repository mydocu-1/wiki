Registers

qa - record macro into register a

@a - run macro from register a

@@ - re-run most recent macro

<register><copying command>, where copying commands are: d c y x etc

"a - store result in register a
"A - append result to register a

"ap - paste content of register a at cursor

Registers:
"" - unnamed default register
"* - system clipboard
"+ - system selection
"0 - last yanked text
"1 .. "9 - last block deletions
"- - last char deletion
"_ - 'black hole'

:reg - content of all registers

<C-r>a - paste from register a in insert mode
